## Usage

    ufo new hook

## Description

Generate hook

## Examples

    $ ufo new hook docker
          create  .ufo/config/hooks/docker.rb
    $ ufo new hook ufo
          create  .ufo/config/hooks/ufo.rb
    $


## Options

```
y, [--force]  # Bypass overwrite are you sure prompt for existing files
```
